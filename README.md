## Description
The Basic Laravel Admin Panel provides an out-of-the-box solution for creating and managing admin panels in Laravel projects. It includes features like user management and permissions.

## Built with
- [Laravel 10](https://github.com/laravel/framework)
- [spatie/laravel-permission](https://github.com/spatie/laravel-permission)
- [Laravel Breeze](https://github.com/laravel/breeze)
- [Bootstrap 5](https://getbootstrap.com/)
- [AdminLTE](https://adminlte.io/)

## Getting Started
To get started, follow these steps:

1. Install [PHP Composer](https://getcomposer.org/).
2. Clone the repository: `git clone https://github.com/Athul-Suresh/default-admin-app.git`
3. Run database migrations: `php artisan migrate`
4. Seed the database: `php artisan db:seed`
5. Install JavaScript dependencies: `npm install`
6. Compile assets: `npm run dev`
7. Start the development server: `php artisan serve`
8. Open your browser and go to http://localhost:8000/
